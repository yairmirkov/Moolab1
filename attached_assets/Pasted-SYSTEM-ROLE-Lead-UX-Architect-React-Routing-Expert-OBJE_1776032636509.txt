Create a new component called AppLayout.tsx.

Create BottomNav.tsx.

🧪 Lab (Routes to the Learning Feed)

🦈 Tank (Routes to the Portfolio/Investing Simulator)

🏦 Vault (Routes to the Moolies Reward Store)

Update the React Router configuration.

Ensure that navigating between these three hubs does not wipe out the user's mooliesBalance or portfolioAssets. These must live in a global React Context or state manager that wraps the entire routing tree, NOT inside the individual tab components.
